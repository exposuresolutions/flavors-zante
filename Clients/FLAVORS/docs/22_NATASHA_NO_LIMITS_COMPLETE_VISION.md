### ?? EMERGENCY CASH SERVICE
**"Lost Your Wallet? We've Got You."**

For customers who lose their wallet, phone, or both:
1. Contact us via borrowed phone or hotel reception
2. Send money via Revolut/Wise to our business account
3. We dispense cash at the bar (minus 5% handling fee)
4. Maximum €500 per transaction
5. ID verification required (passport photo on file or hotel confirmation)

**T&Cs:**
- Service available 6PM - 4AM daily
- 5% handling fee applies
- Customer assumes all transfer fees
- Flavors Zante Ltd not liable for transfer delays
- Photo ID required for collection

## ?? EXPERIENCE AGENT ARMY

### The Commission Model

**Recruitment:**
- Natasha recruits Experience Agents across the island
- Work the strip, beaches, hotels, airports
- Commission-only basis (no risk to us)

**How It Works:**
```
Agent meets customer ? Gives referral code ? Customer books
                                                    ?
                              Lead tracked in GHL CRM
                                                    ?
                              Status updates sent automatically
                                                    ?
                              If sale completes ? Agent gets paid
                              If no sale ? Agent gets nothing
```

**Commission Structure:**
| Package Value | Agent Commission |
|---------------|------------------|
| €0 - €200 | 10% |
| €201 - €500 | 12% |
| €501 - €1000 | 15% |
| €1001+ | 18% |

- Real-time status updates via WhatsApp
- Weekly payment runs (every Friday)
- Full audit trail available
- No wasted time chasing updates
