## ?? THE EXPERIENCE FLOW

### Customer Journey:
```
1. DISCOVER
   - See ad on social media
   - Friend referral
   - Hotel recommendation
   - Walk past Flavors

2. EXPLORE
   - Visit website/Instagram
   - Take "Find Your Party" quiz
   - Browse events by vibe/date/budget

3. MATCH
   - Algorithm suggests best parties
   - Shows who else is going (anonymized)
   - Budget calculator shows options

4. BOOK
   - Select package tier
   - Add extras (transport, food, etc.)
   - Enter referral code (if any)
   - Pay online or at Flavors

5. PREPARE
   - Receive confirmation + QR ticket
   - Get dress code reminder
   - Join WhatsApp group for event
   - Pre-party at Flavors (optional)

6. EXPERIENCE
   - Check-in with QR code
   - Meet matched group (if opted in)
   - Enjoy themed experience
   - Share on social (tagged)

7. EXTEND
   - After-party options
   - Next event invites
   - VIP membership offer
   - Referral incentive

8. REMEMBER
   - Event photos shared
   - Review request
   - Memory video
   - "See you next year" email
```
