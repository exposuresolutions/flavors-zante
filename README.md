## ?? Deployment

This site is deployed via **GitHub Pages** from the `main` branch.

### To Update:
1. Edit files locally or in GitHub
2. Commit changes
3. Push to `main` branch
4. GitHub Pages auto-deploys (usually within 1-2 minutes)

## ?? For GHL Integration

The homepage (`index.html`) is **self-contained** with all CSS and JS embedded inline. 
This allows it to be copied directly into GoHighLevel custom code sections without external dependencies.
