# Exposure OS - Quick Setup

## What's Installed

Your dashboard now has:
- AI Chat Widget with Google Gemini
- Voice Commands
- Project Management UI

## Setup Steps

1. Create file: exposure-dashboard\.env.local
2. Add line: GOOGLE_API_KEY=your_key_here
3. Get key from: https://makersuite.google.com/app/apikey
4. Run: npm install
5. Run: npm run dev
6. Open: http://localhost:3000
